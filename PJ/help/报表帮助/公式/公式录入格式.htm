公式录入格式


公式语法支持下面的符号或值：
数字、字符串、函数、表格单元格名称、数学运算符
1． 数字支持 整数，小数，不支持科学计数法，数值最大支持10亿。
2． 字符串，支持输入字符串以单引号括起来，
3． 数学运算符号支持，简单的四则运算，如 + - * /
    数学运算符号支持下面方法
    1 + 2
    3+10
    A1-210 等写法。
    支持括号: 3 *（3+C10）

4． 支持函数与四则运算及括号混合编写。
如：FS('1243','m','D','','','','','',1)+ FS('1001','m','D','','','','','',1) - FS('1002' ,'m','D','','','','','',1 )

(SFS('1243','m','D','','','','','',1 ) C 100) * 10
