单元格文本格式


　　在单元格格式代码中最多指定四个节。这些节是以分号分隔的，它们顺序定义了格式中的正数、负数、零和文本。如果只指定两个节，则第一部分用于表示正数和零，第二部分用于表示负数。如果只指定一个节，则所有数字都会使用该格式。如果要跳过某一节，则对该节仅使用分号即可。


（1）关于数值的数字格式代码
• # 只显示有意义的数字而不显示无意义的零。
• 如果数字位数少于格式中的零的个数，则 0 （空）将显示无意义的零。
• ? 当设置为等宽字体（例如：Courier New）时，为无意义的零在小数点两边添加空格。可以对具有不等长数字的分数使用 ?。
• % 使用百分数。
• E-、E+、e-和e+ 以科学计数法显示数字。在格式代码中，如果指数代码的右侧含有 0（零）或 #（数字符号），GraspReport 将按科学计数法显示数字，并插入 E 或 e。右侧 0 或 # 的代码个数决定了指数的位数，E- 或 e- 将在指数中填加负号。E+ 或 e+ 在正指数时填加正号，负指数时填加减号。
• $ 贷币符号，在单元格中的数字前￥符号


（2）关于文本和间距的数字格式代码
　　要在单元格中同时显示文本和数字，可将文本字符括在双引号（" "）中，或在单个字符前加反斜线（\）。应将字符放在格式代码的合适部分中。例如，键入格式 $0.00" Surplus";$-0.00" Shortage" 以便将负值显示为“$-125.74 Shortage”。如果是显示这些特殊字符：$ - + / ( ) : ! ^ & '（左单引号） '（右单引号） ~ { } = < > 等，则不需要用引号括起来。

  •  包括文本部分 如果包含文本格式部分，则文本部分总是数字格式的最后一个部分。若要在数字格式中包括文本部分，请在要显示输入单元格中的文本的地方加入符号（@），如果文本部分中没有（@），所输入文本将不会显示出来。如果要一直显示某些带有输入文本的指定文本字符，就应将附加文本用双引号（" "）括起来，例如，"gross receipts for" @。
    在不包含文本部分的格式中，所输入的文本对格式没有影响。
    　

  • 添加空格 如果要在数字格式中创建一个字符宽的空格，请在字符前加上一条下划线（_），例如，在下划线后跟上一个右括号（_)），可使正数和括号内的负数对齐。
    　

  • 重复字符 若要在格式中多次重复一个字符以填满列宽，请在数字格式中加入星号（*）。例如，键入 0*-可在单元格中的数字后重复填加破折号以填充单元格。
