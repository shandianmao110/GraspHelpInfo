GraspReport System中的极限
1、工作表
打开报表数 系统资源限制
工作表大小 系统资源限制，如要导出至EXCEL，则行65536，列256
行高，列宽 500点
单元格文本长度 系统资源限制
工作薄中工作表数 256（包括格式页）
工作薄中颜色 56色自定义调色板
　 　
　

2、输入数据

精度 15-16位
最大正数 1.7 x 10^308
最小负数 -5.0 x 10^C324
最小正数 5.0 x 10^C324
最大负数 -1.7 x 10^308
最长的公式说明 8024字节
选择行列 多行多列（全选为600行，600列）
系统计算的日期基数 1900-1-1
最大时间 9999:99:99
日期录入格式 2004-12-10
时间录入格式 00:23:10

注：报表公式只支持15-16位精度的数字（即-5.0 x 10^C324~1.7 x 10^308），如果数字超出，系统作为文本处理。
