生产模板管理

      如果你经常使用内容相似的组装单、拆分单，可以考虑使用生产模板来进行简化处理。    初次进入生产模板中，系统没有提供任何模板，需要用户创建。在下述例子中，假使你的存货库中有“主机”、“显示器”两种存货，你经常需要将这两种存货组装生成一种叫“计算机”的存货。

使用步骤：

     1、单击菜单栏中“业务处理―库存业务－生产模板”，系统自动弹出“生产模板”。点击“新增”后， 表头上可勾选组装和拆分属性。

     2、录入编号、名称、生成的存货“计算机”，在“该产品由以下存货组成”中录入“主机”，填入“显示器”，以及规格、数量、型号等有关信息。表示组装一台计算机需要主机、显示器各一台。 如果勾选了拆分 ，则需要录入金额拆分比例，金额拆分比例的合计数应为“1”，表示组装一台计算机拆分为一台主机、一台显示器。金额拆分比例的合计数应为“1”。

     4、点击“保存”按钮，保存此模板。

     5、回到模板窗口，可以看见模板已经生成，可以通过点击“修改”、“删除”按钮对其进行修改或删除。 

     6、在使用“组装与拆分单”时，可以“调用模板”，并且录入数量。系统会根据模板内容自动生成单据内容。

     7、模板订单统计，参见下列参考资料。

     8、齐套计算，对选中的存货，根据模板中材料的当前库存计算可以至少组装多少存货。

     9、缺料计算，对选中的存货，手工录入生产数量后，根据模板中材料的当前库存计算可以还缺多少材料。

　

参考资料：  

模板订单统计

　　模板订单统计是针对由生产模板组装而来的存货的统计，例如“联想电脑”由4样配件组成，当销售订单有“联想电脑”100台时，在模板订单统计中可以查询到只针对“联想电脑”的4样配件的库存状况及所需数量， 并且可根据所需配件数量生成进货单或进货订单草稿。

　

表体内容说明：

     配套数量 ：取自模板订单中组装存货所需的配件数；

     账面库存数量：当前库存中的存货数量；

     进货订货数量：指进货订单中未到货数量；

     销售订货数量：指销售订单中未到货数量；

     供应商： 取自基本信息中存货所对应的默认供应商；

     计算数量= 账面库存数量 C 销售订货中未到货数量；

        当计算数量< 0， 说明库存数量不够，需要对配件进行加工。配套存货表体中显示“模板所需数量”一列；

        当计算数量>= 0， 说明库存数量充足，不需要对配件进行加工。配套 存货表体中不显示“模板所需数量”一列；

     可用数量 = 账面库存数量 + 进货订货数量 C 销售订货数量  （进货订货数量、销售订货数量均为未到货数量），可点击“公式”按钮进行修改。

     模板所需数量 = 计算数量 x   配套数量

     剩余数量 = 可用数量 C 模板所需数量

     确认数量：用户填写的需要进货的存货数量；

　

按钮说明：

     修改：修改存货对应的供应商，此处的修改仅是在生成草稿时生效，不会修改基本信息中默认的供应商。

     生成草稿： 根据用户输入的确认数量，生成存货的进货单或进货订单草稿。对应相同供应商的存货将会生成一张草稿。

     公式：可以修改“可用数量”计算的公式。
