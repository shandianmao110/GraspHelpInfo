连接服务器失败的处理方法 

   用户在软件使用过程中，经常会因为各种原因无法连接服务器，这里列出了最常见的几种情况，如果还不能正常连接，请与我们联系。

1、请检查是否安装了服务器端、客户端、SQL-Server2005标准版三个软件。如果没有全部安装，请安装。

2、 检查是否启动了SQL-Server2005，SQL2005正常安装后，将会在开机时，自动启动。手动启动SQL 2005数据时，在开始菜单选择“Microsoft SQL Server 2005-->SQL Server Management Studio”菜单，在“连接到服务器”操作界面中录入登录名和密码以进行连接。

3、是否启动了管家婆服务器支持者，如果启动了，应在Windows桌面右下角状态栏出现类似闪电一样的小图标。同时，管家婆服务器支持者的端口号是否为默认的211。

4、是否正常设置管家婆服务器。运行“开始－>程序－>管家婆软件－>管家婆财贸双全ⅡTop+－服务器”弹出管家婆服务器窗口，点击“设置--设置数据库连接”，在服务器中输入本台计算的名字，在用户名中输入“sa”，在密码处空白（你可以在SQL中进行设置密码）。

注意：在服务器名称处，最好录入SQL 2005实例名，如“RWX-578\SQL2005”。
