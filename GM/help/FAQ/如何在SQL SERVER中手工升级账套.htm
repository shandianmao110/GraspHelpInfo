如何在SQL SERVER中手工升级账套

   有时在进行账套升级时会出现失败的情况，遇到这种情况时若使用数据库维护工具依然不能解决问题，则可以到sql server中的SQL Server Management studio中手工进行账套的升级，具体操作如下：

1、  在开始-程序-microsoft sql server2005中选择 SQL Server Management studio

2、  若安装sql2005时没有设置sa的密码，则不用输入密码直接点击“连接”

 

3、进入后首先点击“文件”按键，选择用于账套升级的sql脚本文件 ，找到后点击“打开”。

4、选择需要进入升级账套对应的数据库

 5、  点击命令栏中的“！执行”开始执行升级语句。

 

6、  执行完毕后，“！执行”将重新出现，而且会返回很执行结果。注：只要所返回的结果中没有红色的文字，则表示升级已成功。

7、退出sql的 SQL Server Management studio，重新进入账套即可。

注意：手工升级补丁的方法同上，只是在选择执行文件时选择升级补丁即可。
