生产模板管理

  如果你经常使用内容相似的组装单、拆分单，可以考虑使用生产模板来进行简化处理。

  初次进入生产模板中，系统没有提供任何模板，需要用户创建。在下述例子中，假使你的存货库中有“主机”、“显示器”两种存货，你经常需要将这两种存货组装生成一种叫“计算机”的存货。

【操作】：业务处理―存货业务―生产模板

功能操作说明：

1、单击”生产模板”菜单，进入生产模板界面。点击“新增”后，在界面 左上角需要选择模板类型。模板有组装和拆分两种类型。

2、如果选择了组装模板，录入编号、名称、生成的 存货“计算机”，在“该产品由以下存货组成”中录入“主机”，填入“显示器”，以及规格、数量、型号等有关信息。如果配套数量选择为1，表示组装一台计算机需要主机、显示器各一台。

3、点击“保存”按钮，保存此模板。

4、回到模板窗口，可以看见模板已经生成，可以通过点击“修改”、“删除”按钮对其进行修改或删除。 

5、如果选择了拆分模板，录入编号、名称、被拆分的存货“生产计算机”，在“该产品拆分成以下存货”中录入“主机”，填入“显示器”，以及规格、数量、型号、金额拆分比例等有关信息。如果配套数量选择为1，表示组装一台计算机拆分为一台主机、一台显示器。金额拆分比例的合计数应为“1”。

6、在使用“组装与拆分单”时，可以“调用模板”，并且录入数量。系统会按照模板所示自动生成有关单据。

【齐套计算】：输入生产数量可以根据库存计算最大成套件的数量

【缺料计算】：输入生产数量，可以根据生产数量计算缺货数量。

【订单统计】：模板订单统计是针对由生产模板组装而来的存货的统计，例如“联想电脑”由4样配件组成，当销售订单有“联想电脑”100台时，在模板订单统计中可以查询到只针对“联想电脑”的4样配件的库存状况及所需数量， 并且可根据所需配件数量生成采购单或采购订单草稿。

表体内容说明：

配套数量 ：取自模板订单中组装存货所需的配件数；

计算数量= 账面库存数量 C 销售订货中未到货数量；

      当计算数量< 0， 说明库存数量不够，需要对配件进行加工。配套存货表体中显示“模板所需数量”一列；

      当计算数量>= 0， 说明库存数量充足，不需要对配件进行加工。配套 存货表体中不显示“模板所需数量”一列；

可用数量 = 账面库存数量 + 采购订货数量 C 销售订货数量  （采购订货数量、销售订货数量均为未到货数量），可点击“公式”按钮进行修改。

模板所需数量 = 计算数量 x   配套数量

剩余数量 = 可用数量 C 模板所需数量

确认数量：用户填写的需要采购的存货数量；

　

【修改】：修改存货对应的供应商，此处的修改仅是在生成草稿时生效，不会修改基本信息中默认的供应商。

【生成草稿】： 根据用户输入的确认数量，生成存货的采购单或采购订单草稿。对应相同供应商的存货将会生成一张草稿。

【公式】：可以修改“可用数量”计算的公式。
